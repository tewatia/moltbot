## üìù Script Format

Each script includes:
- **Voiceover text** (conversational tone)
- **Visual direction** (what to show on screen)
- **Technical steps** (commands & configurations)
- **Production notes** (demos, B-roll, timestamps)

## üé¨ Production Guidelines

### Tone
- Friendly and conversational
- Encouraging for beginners
- Technically accurate for power users

### Visuals
- Clean terminal with large font
- Dark theme preferred
- Notifications disabled during recording

### Pacing
- Allow time for commands to execute on screen
- Pause briefly after important concepts
- Keep energy up throughout
